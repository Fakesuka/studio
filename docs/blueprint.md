# **App Name**: SahaHelper

## Core Features:

- Service Request: Users can request services like car heating, fuel delivery, roadside assistance, and towing.
- Emergency Call: One-click emergency call feature to connect with local emergency services.
- Interactive Map: Display user location and nearby service providers on an interactive map.
- Order Tracking: GPS tracking of service provider in real-time after a service request is accepted.
- Integrated Marketplace: Browse and purchase essential car products such as car blankets, batteries, and anti-freeze.
- Dynamic Pricing: AI tool for calculating dynamic starting prices based on distance, service type, and time of day.
- Offline Mode: Users can create service requests even with a weak signal, to be sent when the connection is restored.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5) to evoke trust and reliability in emergency situations.
- Background color: Light blue (#E3F2FD), a desaturated variant of the primary color.
- Accent color: Royal blue (#64B5F6), an analogous color that provides visual interest and highlights important actions.
- Font: 'Inter', a sans-serif for a modern, machined, objective, and neutral appearance; used for both headlines and body text.
- Use clear and easily recognizable icons to represent service categories and actions.
- Maintain a clean and intuitive layout, with key actions easily accessible.
- Use subtle animations to provide feedback on user interactions and guide attention.
